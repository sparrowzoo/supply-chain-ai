## notebookInstructions
## Notebook 说明

To edit notebook files in the workspace, you can use the edit_notebook_file tool.
要编辑工作区中的 notebook 文件，请使用 edit_notebook_file 工具。

Never use the insert_edit_into_file tool and never execute Jupyter related commands in the Terminal to edit notebook files, such as `jupyter notebook`, `jupyter lab`, `install jupyter` or the like. Use the edit_notebook_file tool instead.
切勿使用 insert_edit_into_file 工具，也不要在终端执行 Jupyter 相关命令（如 `jupyter notebook`、`jupyter lab`、`install jupyter` 等）来编辑 notebook 文件。应始终使用 edit_notebook_file 工具。

Use the run_notebook_cell tool instead of executing Jupyter related commands in the Terminal, such as `jupyter notebook`, `jupyter lab`, `install jupyter` or the like.
运行 notebook 单元格时，也不要在终端执行 Jupyter 相关命令，应使用 run_notebook_cell 工具。

Use the copilot_getNotebookSummary tool to get the summary of the notebook (this includes the list or all cells along with the Cell Id, Cell type and Cell Language, execution details and mime types of the outputs, if any).
如需获取 notebook 摘要（包括所有单元格的 Cell Id、类型、语言、执行信息及输出的 mime 类型等），请使用 copilot_getNotebookSummary 工具。

Important Reminder: Avoid referencing Notebook Cell Ids in user messages. Use cell number instead.
重要提醒：在用户消息中避免引用 Notebook 的 Cell Id，应使用单元格编号。

Important Reminder: Markdown cells cannot be executed
重要提醒：Markdown 单元格无法被执行。
